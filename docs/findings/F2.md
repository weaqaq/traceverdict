# F-2 — 500-character observation truncation invited recovery

On 2026-07-11, tool output shown to the agent was truncated to 500 characters while the unmodified observation remained in the audit payload. Baseline and candidate again had no measurable pass regression on S1–S8 at `k=1` (`alarm=none`).

Trajectories showed why: the agent adapted with narrower commands and many fixture files were already shorter than the limit. A narrower channel is not necessarily a broken channel. The injection was retired from the M1 smoke-test battery and retained as robustness evidence.

**Correction appended 2026-07-12:** the shared S6 failure was a CRLF verifier false positive. Correcting it in both arms does not alter the no-regression conclusion.
